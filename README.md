
# Website Trial

Simple static website built with HTML, CSS and JavaScript.

How to run

- Using XAMPP: place this folder in the `htdocs` directory and open http://localhost/Website-Trial/
- Or open [index.html](index.html) directly in a browser for local testing.

Files

- `index.html` — main page
- `css/style.css` — styles
- `js/main.js` — lightweight interaction (nav, form)
